# Cardápio — Pizza da Villa

Site estático do cardápio da **Pizza da Villa**, pizzaria localizada em Penedo, Itatiaia — RJ.

## Estrutura

```
site-cardapio/
├── index.html          ← página principal
├── css/
│   └── style.css       ← estilos (mobile-first)
└── img/
    ├── logo-black.png  ← logo (fundo preto, blend-mode no site)
    ├── hero-bg.jpg     ← foto de fundo da hero
    └── pratos/         ← fotos dos pratos
```

## Tecnologias

- HTML5 semântico
- CSS3 puro — mobile-first, sem frameworks
- Fontes: [Baloo 2](https://fonts.google.com/specimen/Baloo+2) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts

## Como visualizar

Abra `index.html` diretamente no navegador — não requer servidor.
